CrackTheCampus – Frontend Developer Assignment
🚀 Overview
CrackTheCampus is a platform designed to help students prepare for placements, internships, and competitive aptitude tests.
This project is a React + Tailwind CSS landing page built entirely from scratch, focusing on product design intuition, frontend engineering proficiency, and attention to detail.

🛠️ Tech Stack
React – Component‑based architecture

Tailwind CSS – Utility‑first styling

PostCSS + Autoprefixer – CSS processing

Deployment – Netlify

📱 Features Implemented
Hero Section – Headline, description, CTAs, modern illustration

Features Section – Six core platform capabilities in card layout

Statistics Section – Placement rates, student success metrics

Testimonials Section – Student success stories

Companies Section – Logos of hiring partners

Courses Section – Flagship prep paths

Footer Section – Navigation, contact info, social links

⚙️ Technical Approach
Responsive Design – Mobile‑first, optimized for tablet and desktop

Reusable Components – Each section isolated in src/components

Semantic HTML – Proper use of <header>, <main>, <section>, <footer>

Accessibility – ARIA labels, keyboard navigation

Performance – Lazy‑loaded assets, optimized bundle size

Extra Initiative – Hover states, transitions, dark mode toggle, loading skeletons
